# SESAC 알고리즘 스터디

### 규칙

1. 매일 자정까지 업로드한다. (금요일은 일요일 자정까지)
2. 업로드 안 한 문제당 벌금 5,000원을 부과한다.
3. 검색으로 찾은 풀이는 반드시 자기만의 코드로 다시 써야한다. 그대로 복붙하면 문제당 벌금 10,000원을 부과한다.
4. 화요일마다 오프라인 모의 코딩테스트(알고리즘 3문제, SQL 1문제)를 2시간 동안 실시한다.
