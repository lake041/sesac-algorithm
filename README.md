# SESAC 알고리즘 스터디

### 규칙

1. 매일 두 문제씩 자정까지 업로드한다.
2. 업로드 안 한 문제당 벌금 5,000원을 부과한다.
3. 검색으로 찾은 풀이는 반드시 자기만의 코드로 다시 써야한다. 그대로 복붙하면 문제당 벌금 10,000원을 부과한다.
